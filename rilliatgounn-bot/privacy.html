1. Общие положения

1.1
Настоящая Политика описывает, какие данные собирает и обрабатывает Telegram-бот для помощи учащимся RilliatGounn Bot (далее - «Бот»).
1.2
Бот является некоммерческим учебным проектом, разработанным в образовательных целях.

2. Какие данные собираются

2.1
Бот может собирать и обрабатывать следующие данные:
  • Telegram ID пользователя;
  • имя пользователя (username);
  • сообщения, отправленные боту;
  • технические данные, необходимые для работы бота;
  • данные электронного журнала ЭлЖур (при добровольном подключении);
  • настройки пользователя внутри бота.

3. Цели обработки данных

3.1
Данные используются исключительно для:
  • обеспечения работы функционала Бота;
  • корректной обработки команд;
  • предоставления ответов AI-ассистента;
  • ведения статистики и логирования;
  • предотвращения злоупотреблений.

4. Хранение данных

4.1
Данные хранятся:
  • в базе данных Бота (SQLite);
  • во временном хранилище (Redis).
4.2
Все данные пользователей хранятся на серверах, расположенных на территории Российской Федерации, в соответствии с требованиями законодательства РФ.
4.3
Данные не используются в коммерческих целях.

5. Передача данных третьим лицам

5.1
Часть данных (тексты запросов к AI) может передаваться сторонним AI-провайдерам для генерации ответов.
5.2
Передача данных осуществляется только в объёме, необходимом для работы AI.

6. Защита данных

6.1
Разработчик принимает разумные меры для защиты данных от несанкционированного доступа.
6.2
Несмотря на это, абсолютная безопасность данных не может быть гарантирована.

7. Удаление данных

7.1
Пользователь вправе запросить удаление своих данных, обратившись через команду /support.
7.2
После удаления данных дальнейшее использование Бота станет невозможным до момента повторной регистрации.

8. Изменение политики

8.1
Политика конфиденциальности может быть изменена без предварительного уведомления.
8.2
Актуальная версия всегда доступна пользователю через Бота.

9. Контакты

9.1
По вопросам обработки данных пользователь может обратиться через команду /support.
